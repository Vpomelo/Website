Topics

We hope that the proposed workshop is fruitful in attracting more attention to developing more data mining algorithms and models for scientific education. We encourage submissions related to data driven research on learning science and education system, including but not limited to:

  • Data mining in education
  • Data driven learning analytics
  • Data mining for computer-aided education
  • Data mining in online learning platform
  • Data mining for personalized guided learning
  • Brain imaging computation for cognitive in learning science
  • Natural language processing for educational data analysis
  • Educational image processing for computer-aid tools
  • Multi-model data mining and multi-source data mining for personalized learning
  • Graph reconstruction and representation for DMLS
  • Heterogeneous information network representation learning for DMLS
  • Small sample issues in education data mining (EDM)
  • Distributed and privacy-protected data mining for DMLS
  • Federated learning for education
  • Fair machine learning and explainable data mining models for EDM
  • Knowledge graph construction and learning for EDM
  • Big data driven education theory and educational psychology research
